## Variabel
Variabel adalah sebuah lokasi di memori dimana seorang Programmer dapat menyimpan sebuah nilai. Dalam Python deklarasi variabel tidak dilakukan secara eksplisit. 
#### Contoh Kode 1.1
```python
height = 1.79
weight = 68.7

bmi = weight / height ** 2
print(bmi)
```
